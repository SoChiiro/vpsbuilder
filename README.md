# VPS BUILDER

## Project setup

```
git clone +url
```
Once git cloned you run the following command in your terminal :

```
npm install
```

### Start the server
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
